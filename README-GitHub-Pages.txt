GitHub Pages公開手順

1. このZIPを展開する
2. 展開後の中にある index.html / assets フォルダ / dict フォルダ を確認する
3. GitHub の lyric-memo リポジトリを開く
4. 既存の index.html があれば削除するか上書きする
5. Add file → Upload files を押す
6. 展開した中身をまとめてアップロードする
   - index.html
   - assets フォルダ
   - dict フォルダ
7. Commit changes を押す
8. Settings → Pages で Branch が main / (root) になっているか確認する
9. 数分待って公開URLを開く
10. iPhone の Safari で開いて、共有 → ホーム画面に追加

メモ
- 初回は辞書の読み込みで少し時間がかかります
- 「好き」で保存した内容を「すき」で検索できるようになります
- データは localStorage に保存されます


アップロード後のURL例
- https://あなたのユーザー名.github.io/リポジトリ名/

補足
- GitHub PagesはリポジトリのHTML/CSS/JavaScriptをそのままWebサイトとして公開できます。
- iPhoneではSafariでURLを開き、共有→ホーム画面に追加→Webアプリとして開く をオンにします。
